O que é JavaScript

O JavaScript é uma linguagem de programação que permite criar interatividade e dinamismo em páginas web. Ele é usado para fazer com que a página "reaja" às ações do usuário, como cliques em botões, envio de formulários e animações.

Pense no JavaScript como o cérebro da página. Ele adiciona comportamento, como carregar conteúdo dinamicamente ou validar dados, tornando a experiência do usuário mais rica e fluida.
