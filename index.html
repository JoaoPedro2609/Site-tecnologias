O que é HTML

O HTML (HyperText Markup Language) é a linguagem de marcação padrão para a criação de páginas web. Ele define a estrutura e o conteúdo de um site, organizando elementos como títulos, parágrafos, listas e imagens.

Pense no HTML como o esqueleto de uma página. Ele não cuida do visual, mas sim da organização lógica do que será exibido. Sem ele, a internet como conhecemos não existiria.
